# Prompt-Para-ChatGPT
Lo primero es saber, que es un Prompt, pues bien cuando hablamos de un prompt hablamos de la instrucción o mensaje que le das a un modelo de inteligencia artificial (o modelo de Lenguaje) para indicarle qué debe hacer y cómo debe responder. Puede ser una pregunta, una descripción, una orden o un conjunto de reglas. Su función es guiar el comportamiento de la IA proporcionando contexto, objetivo, estilo y límites, para así obtener un resultado específico y coherente.
Asi que aquí ire colgando diferentes promts para diferentes situaciones.
